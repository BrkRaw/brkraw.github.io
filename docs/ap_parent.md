## Python API
- import module
```js
>>> import brkraw as br
```

- load dataset
```js
>>> rawdata = br.load(<PATH>)
```

- If you want to learn more detail about Pythop API, please check our example
[Jupyter Notebooks](https://github.com/brkraw/bruker/blob/master/examples/BrkRaw_PythonAPI.ipynb).
